# 一些小脚本

| 名称             | 功能                                             |
| ---------------- | ------------------------------------------------ |
| 自动拨号         | 保持网络连接正常，可简单修改使用多个账号         |
| 无黑窗口静默执行 | 在 shell 指令执行时隐藏黑窗口                    |
| 百度资源收录提交 | 将网络链接提交到百度资源统计中加快收录           |
| 数据库填充       | 用于将 markdown 文件内容填入 2021 版博客的数据库 |
